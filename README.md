# HOTEL_MANAGMENT_SYSTEM

A beginner-friendly Hotel Management System built using Python and Object-Oriented Programming (OOP) concepts such as Classes, Inheritance, Encapsulation, Polymorphism, and Abstraction.

🏨 Hotel Management System

**📖 Project Description**

The Hotel Management System is a console-based Python project designed to manage hotel operations efficiently using Object-Oriented Programming (OOPs) concepts.
This project helps beginners understand how real-world systems can be developed using Python classes and objects.

The system allows users to:

Add customer details
Book hotel rooms
Check room availability
Generate bills
Manage customer records
Perform menu-driven operations

This project is ideal for students and beginners who want practical implementation of OOP concepts in Python.

**✨ Features**

✅ Room Booking System
✅ Customer Management
✅ Room Availability Check
✅ Bill Generation
✅ Menu-Driven Interface
✅ Games menu & other activities 
✅ File Handling Support (Optional)
✅ Easy-to-Understand Code Structure
✅ Beginner-Friendly Project

**🧠 OOP Concepts Used**

OOP Concept	Usage in Project
Classes & Objects	Used to create hotel, customer, and room entities
Inheritance	Reusing properties and methods from base classes
Encapsulation	Protecting data using private/protected members
Polymorphism	Method overriding and flexible functionality
Abstraction	Hiding implementation details using methods/classes
Constructor (__init__)	Initializing object attributes
